# ربات دانلود فایل تلگرام

یک ربات تلگرام که فایل‌ها را از URLها دانلود کرده و به کاربران تحویل می‌دهد، برای کار در محیط‌های شبکه محدود مانند ایران طراحی شده است.

## ویژگی‌ها

- **رابط کاربری با انتخاب کاربر**: کاربران می‌توانند روش دانلود مورد نظر خود را انتخاب کنند
- **روش‌های متعدد دانلود**:
  - 📱 **دانلود تلگرام**: انتقال مستقیم فایل از طریق تلگرام (فایل‌های کمتر از ۵۰ مگابایت)
  - 📦 **تقسیم تلگرام**: فایل‌های بزرگ به بخش‌ها تقسیم می‌شوند (۵۰-۵۰۰ مگابایت)
  - 🐙 **ذخیره گیت‌هاب**: فایل‌ها در ریپازیتوری گیت‌هاب با دسترسی URL خام ذخیره می‌شوند (در ایران کار می‌کند!)
  - 🌐 **سرور HTTP**: لینک دانلود مستقیم از VPS (برای فایل‌های بسیار بزرگ)
- **مدیریت هوشمند فایل**: تشخیص خودکار اندازه فایل و پیشنهاد روش مناسب
- **تاب‌آوری شبکه**: دور زدن محدودیت‌های شبکه ایران
- **مبتنی بر VPS**: روی VPS در منطقه بدون محدودیت اجرا می‌شود
- **یکپارچگی گیت‌هاب**: استفاده از تحویل محتوای خام گیت‌هاب

## معماری

ربات روی یک VPS در منطقه بدون محدودیت (دور از سانسور اینترنت) اجرا می‌شود و:
1. URLهای دانلود را از کاربران از طریق تلگرام دریافت می‌کند
2. فایل‌ها را از اینترنت بدون محدودیت دانلود می‌کند
3. فایل‌ها را از طریق تلگرام ارسال می‌کند یا لینک‌های دانلود جایگزین ارائه می‌دهد
4. فایل‌های بزرگ را با تقسیم یا ارائه لینک‌های دانلود مستقیم مدیریت می‌کند

## راه‌اندازی

### پیش‌نیازها

- پایتون ۳.۸+
- توکن ربات تلگرام (از @BotFather)
- VPS در منطقه بدون محدودیت (هر کشوری بدون سانسور اینترنت)

### نصب

```bash
# کلون کردن ریپازیتوری
git clone <repo-url>
cd telegram-file-bot

# نصب وابستگی‌ها
pip install -r requirements.txt

# پیکربندی متغیرهای محیطی
cp .env.example .env
# .env را با توکن ربات و تنظیمات خود ویرایش کنید
```

### پیکربندی

فایل `.env` را ویرایش کنید:
```
TELEGRAM_BOT_TOKEN=توکن_ربات_شما
ADMIN_USER_ID=آیدی_تلگرام_شما
MAX_FILE_SIZE_MB=50
TEMP_DOWNLOAD_DIR=/tmp/telegram_bot_downloads
ENABLE_HTTP_SERVER=True
HTTP_SERVER_PORT=8080
VPS_HOST=آی_پی_یا_دامنه_VPS_شما
GITHUB_TOKEN=توکن_گیت‌هاب_شما
GITHUB_REPO=نام_کاربری/نام_ریپازیتوری
```

### اجرا

```bash
# اجرا مستقیم
python bot.py

# یا استفاده از سرویس systemd (توصیه می‌شود برای VPS)
sudo cp telegram-file-bot.service /etc/systemd/system/
sudo systemctl daemon-reload
sudo systemctl enable telegram-file-bot
sudo systemctl start telegram-file-bot
```

## نحوه استفاده

1. یک URL دانلود به ربات ارسال کنید
2. ربات فایل را روی VPS دانلود می‌کند
3. **روش دانلود خود را انتخاب کنید**:
   - 📱 **دانلود تلگرام**: انتقال مستقیم فایل (سریع، فایل‌های <۵۰ مگابایت)
   - 📦 **تقسیم تلگرام**: فایل‌های بزرگ به بخش‌ها تقسیم می‌شوند (۵۰-۵۰۰ مگابایت)
   - 🐙 **ذخیره گیت‌هاب**: فایل در گیت‌هاب با URL خام ذخیره می‌شود (در ایران کار می‌کند!)
   - 🌐 **سرور HTTP**: لینک دانلود مستقیم از VPS (برای فایل‌های بسیار بزرگ)
4. **روش گیت‌هاب** (توصیه می‌شود برای ایران):
   - فایل در ریپازیتوری گیت‌هاب شما ذخیره می‌شود
   - شما یک URL خام گیت‌هاب دریافت می‌کنید
   - دانلود مستقیم از گیت‌هاب (در ایران کار می‌کند)
   - بدون محدودیت اندازه برای ذخیره‌سازی گیت‌هاب

## روش‌های دور زدن شبکه

### راه‌اندازی فعلی
- **تونل‌سازی DNS**: کند اما برای دسترسی تلگرام کار می‌کند
- **فایل‌های کد گیت‌هاب**: قابل دسترسی، ریلیزها مسدود هستند
- **VPS در منطقه آزاد**: دسترسی کامل اینترنت

### روش‌های جایگزین
1. **رله HTTP**: استفاده از سرویس‌های رله HTTP برای ترافیک HTTP
2. **VPN DNS**: استفاده از VPN مبتنی بر DNS برای دسترسی تلگرام
3. **URLهای خام گیت‌هاب**: استفاده از گیت‌هاب به عنوان CDN برای فایل‌های کوچک
4. **دانلود مستقیم VPS**: کاربران می‌توانند مستقیماً از سرور HTTP VPS دانلود کنند

## مدیریت اندازه فایل

- **<۵۰ مگابایت**: ارسال مستقیم از طریق تلگرام
- **۵۰-۵۰۰ مگابایت**: تقسیم به بخش‌ها و ارسال از طریق تلگرام
- **>۵۰۰ مگابایت**: ارائه لینک دانلود مستقیم از سرور HTTP VPS
- **بسیار بزرگ**: استفاده از GitHub LFS یا ذخیره‌سازی جایگزین

## امنیت

- حالت فقط ادمین (قابل تنظیم)
- محدودیت اندازه فایل
- اعتبارسنجی URL
- پاکسازی فایل‌های موقت
- محدودیت نرخ (اختیاری)

## عیب‌یابی

### ربات پاسخ نمی‌دهد
- توکن ربات صحیح است را بررسی کنید
- دسترسی اینترنت VPS را تأیید کنید
- لاگ‌ها را بررسی کنید: `journalctl -u telegram-file-bot -f`

### شکست فایل‌های بزرگ
- تایم‌اوت را در تنظیمات افزایش دهید
- فضای دیسک VPS را بررسی کنید
- از روش دانلود جایگزین استفاده کنید

### مشکلات شبکه
- اطمینان حاصل کنید تلگرام از طریق VPN شما قابل دسترسی است
- تنظیمات فایروال VPS را بررسی کنید
- کارکرد تونل‌سازی DNS را تأیید کنید
